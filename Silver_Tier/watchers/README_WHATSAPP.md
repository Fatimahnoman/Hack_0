### Or Directly

```powershell

```

---

## 📋 First Time Setup

### Step 1: Install Dependencies

```powershell
pip install playwright
playwright install chromium
```

### Step 2: Run the Watcher

```powershell
python watchers/whatsapp_watcher.py
```

### Step 3: Login to WhatsApp

- Browser window will open
- QR code will appear
- Scan with your phone:
  - Open WhatsApp on phone
  - Settings → Linked Devices → Link a Device
  - Scan QR code

### Step 4: Session Saved

After first login, session is saved to:
```
E:\Hackathon_Zero\Silver_Tier\sessions\whatsapp_session\
```

Next time, no QR code needed!

## 🧪 Test It

### Test 1: Send Yourself a Message

1. Run the watcher
2. From another number, send:
   ```
   Hey! Can you send me the invoice ASAP?
   ```
3. Wait 60 seconds
4. Check `Vault/Needs_Action/` for new `.md` file

### Test 2: Keyword Detection

**High Priority Keywords:**
- invoice
- payment
- urgent
- asap
- help

**Medium Priority Keywords:**
- price
- quote
- hello
- hi
- hey

### Test 3: Check Output

```powershell
# List new files
dir Vault\Needs_Action\WHATSAPP_*.md

# View file content
type Vault\Needs_Action\WHATSAPP_*.md
```

## 🐛 Troubleshooting

| Issue | Fix |
|-------|-----|
| `Playwright not installed` | Run: `pip install playwright && playwright install chromium` |
| `QR code not showing` | Close browser, delete `sessions/whatsapp_session`, run again |
| `No files created` | Check keywords in message, wait 60 seconds |
| `Session expired` | Watcher will auto-reconnect, scan QR again |
| `Chrome crash` | Close all Chrome windows, run again |
